ISP & Network Device Monitoring

Pre-built dashboards for Tier-1 ISP operations and enterprise network hardware.

Network Hardware Vendor

SNMP Exporter Setup Guide

Add devices to inventory and generate configs:

# Add a Cisco router
python setup/scripts/setup_network_devices.py add \
  --vendor cisco \
  --name core-router-01 \
  --host 10.0.0.1 \
  --community readonly \
  --role pe

# Add a Juniper PE router
python setup/scripts/setup_network_devices.py add \
  --vendor juniper \
  --name pe-01 \
  --host 10.0.1.1 \
  --version v3 \
  --v3-username monitor

# Generate SNMP exporter + Prometheus configs
python setup/scripts/setup_network_devices.py generate

Device management commands:

# List all devices
python setup/scripts/setup_network_devices.py list

# Test ICMP reachability
python setup/scripts/manage_devices.py ping

# Test SNMP connectivity
python setup/scripts/manage_devices.py snmp-test

# Export inventory to CSV
python setup/scripts/manage_devices.py export --format csv

# Provision all ISP dashboards to Grafana
python setup/scripts/provision_dashboards.py manifest isp_manifest.yml

Multi-Cloud Monitoring

Configure datasources and deploy dashboards for AWS, GCP, and Azure.

☁️ AWS CloudWatch Setup

Or use CLI to configure:

python setup/scripts/setup_cloud_providers.py aws \
  --region us-east-1 \
  --access-key AKIA... \
  --secret-key ...

# Or with IAM Role assumption
python setup/scripts/setup_cloud_providers.py aws \
  --region us-east-1 \
  --role-arn arn:aws:iam::123456789:role/GrafanaMonitoring

# Required IAM permissions:
#   cloudwatch:GetMetricData
#   cloudwatch:ListMetrics
#   ec2:DescribeInstances
#   tag:GetResources

🟡 GCP Cloud Monitoring Setup

CLI setup:

python setup/scripts/setup_cloud_providers.py gcp \
  --project my-gcp-project \
  --key-file /path/to/service-account.json

# Required roles:
#   roles/monitoring.viewer
#   roles/compute.viewer (for GCE labels)
#   roles/container.viewer (for GKE)

🔷 Azure Monitor Setup

CLI setup:

python setup/scripts/setup_cloud_providers.py azure \
  --tenant-id <tenant-id> \
  --client-id <client-id> \
  --client-secret <secret> \
  --subscription-id <sub-id>

# App Registration required permissions:
#   Microsoft.Insights/*/read
#   Microsoft.Monitor/*/read
